# Security Policy

## Supported Versions

The currently supported major line is:

- `1.x` (Ibexa 4.6, Symfony 5.4 LTS)

## Reporting a Vulnerability

Please do not report security vulnerabilities in public GitHub issues.

Report privately to:

- `sklimaszewski@kaliop.com`

Include as much detail as possible:

- affected version
- vulnerability type and impact
- reproduction steps
- proof of concept (if available)
- possible remediation ideas

You will receive an acknowledgment as soon as possible. We will triage, confirm
impact, and coordinate a fix and release.
